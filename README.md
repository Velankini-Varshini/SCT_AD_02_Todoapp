#📝 Todo App

A modern Todo application built with **React Native** and **TypeScript**. It allows users to create, manage, and organize daily tasks with priority levels, due dates, theme switching, and task editing.

---

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 📅 Add due dates
- 🔥 Priority selection (High, Medium, Low)
- 🌙 Light & Dark mode
- 📱 Responsive and clean UI

---

## 🛠️ Built With

- React Native
- TypeScript
- React Hooks
- FlatList
- React Native Vector Icons

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/<your-username>/<repo-name>.git
```

### Navigate into the project

```bash
cd <repo-name>
```

### Install dependencies

```bash
npm install
```

### Start Metro

```bash
npm start
```

### Run on Android

```bash
npm run android
```

### Run on iOS

```bash
npm run ios
```
